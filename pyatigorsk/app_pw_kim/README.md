# Конвертер единиц длины с алгоритмом сортировки на линейных сетях

## Описание
Данный проект представляет собой веб-приложение для конвертации различных единиц измерения длины с использованием алгоритма сортировки на линейных сетях. Конвертер поддерживает широкий спектр единиц измерения, включая метрические, астрономические, китайские, японские, английские/американские, морские и старорусские меры длины.

## Алгоритм сортировки на линейных сетях
В проекте используется алгоритм сортировки на линейных сетях (Linear Network Sorting), который имеет следующие особенности:

1. **Принцип работы:**
   - Алгоритм создает сеть компараторов, где каждый компаратор сравнивает и при необходимости меняет местами два соседних элемента
   - Сеть компараторов фиксирована и не зависит от входных данных
   - Все сравнения выполняются в предопределенном порядке

2. **Реализация:**
   ```javascript
   function sortNetwork(arr) {
       const n = arr.length;
       const network = [];
       const sortedArr = [...arr];
       
       // Создание сети компараторов
       for (let i = 0; i < n - 1; i++) {
           for (let j = 0; j < n - 1 - i; j++) {
               network.push([j, j + 1]);
           }
       }
       
       // Применение сортировочной сети
       network.forEach((comparison) => {
           const [i, j] = comparison;
           if (sortedArr[i] > sortedArr[j]) {
               [sortedArr[i], sortedArr[j]] = [sortedArr[j], sortedArr[i]];
           }
       });
       
       return sortedArr;
   }
   ```

3. **Особенности реализации:**
   - Создание сети компараторов происходит один раз при каждой сортировке
   - Каждый компаратор представлен парой индексов [i, j]
   - Сравнения выполняются последовательно для каждой пары индексов
   - Используется современный синтаксис JavaScript для обмена значений

## Применение в конвертере
Алгоритм используется для упорядочивания результатов конвертации по следующему принципу:

1. Все значения конвертируются в метры (базовая единица)
2. Затем выполняется конвертация из метров во все поддерживаемые единицы
3. Полученные значения сортируются с помощью алгоритма линейных сетей
4. Отсортированные результаты отображаются в виде карточек

## Структура проекта
- `index.html` - основной HTML файл с разметкой интерфейса
- `styles.css` - стили для оформления интерфейса
- `script.js` - JavaScript код с реализацией алгоритма и логики конвертации

## Использование
1. Откройте `index.html` в веб-браузере
2. Введите числовое значение в поле ввода
3. Выберите исходную единицу измерения из выпадающего списка
4. Результаты конвертации во все поддерживаемые единицы будут автоматически отображены и отсортированы

## Преимущества использования алгоритма линейных сетей
- Предсказуемое время выполнения
- Фиксированное количество сравнений
- Возможность параллельного выполнения (в данной реализации не используется)
- Простота реализации и понимания алгоритма
